* Внешние обработки для получения остатков Номенклатуры.
*
* Файл <ОстаткиНоменклатуры.epf> предназначен 
* Для вывода результатов из справочника <Номенклатура>
* С количеством товара на текущий момент времени
* То есть данная внешняя обработка позволяет
* Отслеживать остатки товаров в режиме
* Реального времени. Подходит для разничной
* Торговли без учета по складам или магазинам.
* Подходит для количественного учета.
*
* Файл <ОстаткиНоменклатурыПоСкладам.epf> предназначен
* Для вывода количества остатков товара
* Из справочника <Номенклатура> с разбивкой по складам.
* То есть, при активации строки справочника <Номенклатура>
* Пользователь получает таблицу с информацией
* Сколько и на каком складе имеется данный товар
